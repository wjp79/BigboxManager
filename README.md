## About BigBoxManager

BigBoxManager grew out of my hobby for retro big box games, collecting, reselling, and building useful tools.
I made this app because I wanted something that really matched the way I work: a practical, local Windows app for keeping track of sets, individual titles, purchase costs, asking prices, sales, and collection value.
The app has been in development for several years as a personal tool for my own collection and trading workflow. After seven earlier versions and many iterations, BigBoxManager has now reached version 8. This current version feels ready to be shared publicly for the first time.
BigBoxManager is still shaped by real use, testing, mistakes, fixes, and new ideas. It is not a commercial product, but a hobby project that has grown into something others might find useful too.
If the app is useful to you and you enjoy using it, feel free to leave a comment. Feedback is always welcome!
